# Step 4: App Icons - AI Mockup Prompts

## Main Screen Prompt

Create a mobile screen for icon upload and processing:

**Screen**: App Icons (Step 4 of 10)

**Layout**:
1. Header with step indicator
2. Upload zone (drag-drop)
3. Shape selector (circle, rounded, squircle, square)
4. Background color picker
5. Generated icons grid with platform tabs
6. Navigation buttons

**Style**: Material Design 3, blue primary (#2563eb)

---

## Component Prompts

### Upload Zone
- Dashed border, rounded corners
- File icon centered
- "Drag & drop or click" text
- Accepted formats: PNG, SVG, 1024x1024+

### Shape Selector
- 4 shape options in a row
- Visual icons for each shape
- Selected state with blue border

### Icon Grid
- Platform tabs (Android, iOS, Web)
- Icons displayed at actual sizes
- Labels showing density/size
