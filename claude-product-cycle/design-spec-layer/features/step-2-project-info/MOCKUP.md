## Validation States

### Real-time Validation

```
User types: "com.example"
Status: Valid (no error shown)

User types: "invalid package"
Status: Error shown immediately
Message: "Package name must be valid (e.g., com.example.app)"
```

### Form Submission Validation

```
Missing required field:
- Field highlighted with red border
- Error message below field
- Focus moves to first error
- Error summary at top (optional)
```
