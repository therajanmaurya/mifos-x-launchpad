## Interactions

### Color Swatch Click
- Opens color picker popover
- Position: Below swatch, left-aligned
- Animation: Fade in, 150ms

### Preset Card Click
- Applies preset colors
- All color swatches update
- Preview updates
- Animation: 200ms transition

### Dark Mode Toggle
- Enables/disables dark mode section
- When enabled, show dark color pickers
- Animation: Slide down, 200ms

### Preview Mode Toggle
- Switches between light/dark preview
- Animation: Crossfade, 200ms
