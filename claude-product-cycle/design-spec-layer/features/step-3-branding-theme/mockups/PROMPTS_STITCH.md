## Main Screen Prompt

Create a mobile screen for theme customization:

**Screen name**: Branding & Theme
**App type**: Project generator wizard

The screen should have:
1. Header with back button and "Step 3 of 10"
2. Title "Branding & Theme"
3. Grid of 6 theme preset cards (colored squares with names)
4. Color picker inputs with swatches for primary, secondary, accent, etc.
5. Dark mode toggle switch
6. Live preview showing a phone with the selected colors
7. Previous and Next buttons

**Style**:
- Clean Material Design 3
- Color swatches as colored squares
- One preset shown as selected with checkmark
- Phone mockup for preview
