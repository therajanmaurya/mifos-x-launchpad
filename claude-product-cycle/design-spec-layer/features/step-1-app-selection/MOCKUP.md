## Interactions

### Selection Animation

```css
.app-card {
  transition: all 200ms ease-in-out;
}

.app-card:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.app-card.selected {
  animation: pulse 300ms ease-out;
}

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.02); }
  100% { transform: scale(1); }
}
```

### Checkmark Animation

```css
.checkmark {
  animation: checkmark-appear 200ms ease-out;
}

@keyframes checkmark-appear {
  0% {
    opacity: 0;
    transform: scale(0);
  }
  100% {
    opacity: 1;
    transform: scale(1);
  }
}
```
